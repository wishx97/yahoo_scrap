## Getting Started
### Installation
1. Clone the repo
```sh
git clone https://github.com/wishx97/yahoo_scrap.git
```
2. Install required package
```sh
pip install -r requirements.txt
```
3. Run the script
```sh
scrapy crawl yahoo
```
### Result
Result will be exported as items.json under "results" directory
